# Junior Fullstack-аналитик — решение задания

## Цель
Подготовить аналитические артефакты для разработки функционала
(фронтенд + бэкенд) в рамках учебного кейса.

## Контекст
Рассматривается система с клиентской частью и серверной логикой.
Задача аналитика — формализовать требования, архитектурные решения
и взаимодействие компонентов.

## Что сделано
- Проанализированы требования задания
- Подготовлены системные требования
- Описана архитектура решения
- Сформирована спецификация API
- Зафиксированы ключевые сущности и взаимодействия

## Структура репозитория
- `requirements/` — системные и функциональные требования
- `architecture/` — архитектурная схема решения
- `api/` — описание API и взаимодействий

## Как проверять
Файлы структурированы по папкам.
Каждый документ отражает отдельный аналитический артефакт.
